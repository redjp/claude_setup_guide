# 情報の扱いとプラン

Claude に入力した情報がどう扱われるか、プランによる違いをまとめます。**機密情報を扱う前に**目を通してください。

!!! warning "最新の正確な情報は公式で"
    データの取り扱い・保存期間・料金・プラン内容は**変わることがあります**。重要な判断の前に、必ず[プライバシーセンター](https://privacy.claude.com/)・[ヘルプセンター](https://support.claude.com/)で最新を確認し、社内の IT・法務にもご相談ください。本ページは執筆時点の概要です。

## 入力した情報はどう扱われる?

- **処理はクラウドで** … 入力内容は Anthropic のサーバーで処理されます（端末内だけで完結しません）。
- **第三者に販売されない** … ユーザーのデータが第三者に売られることはありません。
- **機微データのフィルタリング** … 学習に使う場合でも、ユーザーID（メール等）と切り離し、機微なデータをフィルタ／難読化する仕組みがあります。

## モデルの学習（改善）に使われる?

| プラン | 学習への利用（既定） |
| --- | --- |
| **Free / Pro / Max（個人向け）** | **オプトイン制**。設定「AIモデルの改善にご協力ください」を**オンにしたときだけ**学習に使われます（既定はオフ）。 |
| **Team / Enterprise（ビジネス向け）** | 商用利用規約のもと、**既定で学習に使われません**。 |

- 会話を**削除**すれば、その会話は今後の学習に使われません。
- **シークレット（Incognito）チャット**は、設定に関わらず学習に使われません。

## 保存期間（個人向けプラン）

| 状況 | 保存期間 |
| --- | --- |
| 学習を**オフ**にして会話を削除 | 履歴から即時、バックエンドから **30日以内**に削除 |
| 学習を**オン**にしている | 学習用に**最長5年**（識別情報を外した形）保持され得る |
| ポリシー違反として**フラグ**された内容 | 入出力を**最長2年**、判定スコアを最長7年 |
| 送信したフィードバック | 5年 |

## 会話を誰かが見ることはある?

- **学習オン**の場合、限られた担当者がモデル改善のために閲覧することがあります（ユーザーIDと切り離し済み）。
- **安全性・ポリシー違反**としてフラグされた会話は、設定に関わらず安全目的で確認されることがあります。

## 機密情報の入力について

保護の仕組みはありますが、**パスワード・口座番号・医療情報・社外秘文書などの極めて機密性の高い情報は、設定に関わらず入力しない**ことが推奨されています。**「学習オフ＝安全」ではなく**、「学習に使われない」という意味にすぎません（[おすすめ設定のプライバシー](settings.md) も参照）。

## プランの違い（概要）

| 項目 | Free | Pro | Max | Team | Enterprise |
| --- | :--: | :--: | :--: | :--: | :--: |
| 利用量 | 少 | 標準 | 大 | Pro の 1.25〜6.25倍/セッション ※ | 大（カスタム） |
| 学習（既定） | オプトイン | オプトイン | オプトイン | 使わない | 使わない |
| 管理者機能（請求・権限・上限） | − | − | − | ○ | ○ |
| SSO / ドメイン管理 | − | − | − | ○ | ○ |
| データ保持のカスタム設定 | − | − | − | − | ○ |
| 監査ログ・Compliance API | − | − | − | − | ○ |
| 規模 | 個人 | 個人 | 個人 | 5〜150席 ※ | 大規模 |

※ 利用量・席数・料金は変わります。最新は[公式の料金ページ](https://www.anthropic.com/pricing)でご確認ください。

## 機密を本格的に扱う組織には

- **Team / Enterprise プラン** … 既定で学習に使われず、管理者制御・SSO に対応。**Enterprise** はさらに**データ保持のカスタム設定**・監査ログ・Compliance API など、組織のコンプライアンス要件に対応できます。
- **API のゼロデータ保持** … 開発用途で、要件を満たせば入出力を保持しない構成も可能です。
- 御社の規程に合うかは、**公式のデータ取扱い文書＋自社の IT・法務**で判断してください。
