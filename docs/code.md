# Code 機能を使う（Desktop 内）

上部タブの **「Code」** では、Claude Desktop の中で **コードの作成・修正やファイル操作、定型作業の自動化** などの実作業を任せられます。プログラミングだけでなく、「フォルダ内のファイルを整理する」「データを変換する」といった作業にも使えます。

!!! info "まずは Chat / Cowork で十分なことも多い"
    文章作成やデータ整理だけなら [Chat](quickstart.md) や [Cowork](cowork.md) で足ります。Code 機能は、**ファイルやコードを直接さわる作業**が必要なときの選択肢です。

## Code 機能とは / 始め方

1. 上部タブの **「Code」** を開く
2. 作業対象の **フォルダ（プロジェクト）** を指定する
3. やってほしいことを言葉で依頼する（例：「このフォルダの請求書PDFから金額を抜き出して一覧にして」）

!!! note "🖼 画面イメージ（後日挿入予定）"
    Code タブの画面と、フォルダ指定の様子。

## モード切替（自動モード） { #auto-mode }

Code 機能では、Claude に **「どこまで自動で実行させるか」** を切り替えられます。これが**安全と効率のバランス**を決める重要な設定です。

| モード | 動き | 向いている場面 |
| --- | --- | --- |
| **都度確認（既定）** | 操作のたびに「実行してよいか」を確認 | 初心者・大事な環境。**まずはこれ** |
| **自動モード** | 確認なしで連続して実行 | 内容を把握できる・信頼できる作業 |

!!! warning "自動モードの注意"
    自動モードは、**ファイルの変更やコマンドの実行を確認なしで**行います。速くて便利ですが、内容を把握できる作業・信頼できる範囲だけで使ってください。重要なファイルがある環境では、まず**都度確認**で様子を見るのがおすすめです。

!!! note "🖼 画面イメージ（後日挿入予定）"
    モード切替メニュー（都度確認／自動）。

## 基本の使い方と活用例

- 「この CSV を、月別の集計表に変換して」
- 「フォルダ内のファイル名を、`日付_案件名` の形式にそろえて」
- 「議事録のテキストから、ToDo を抜き出して一覧ファイルにして」

!!! tip "もっと本格的に使うなら"
    Code 機能で物足りなくなったら、ターミナルで動く本格版 **[Claude Code CLI](cli.md)** に進めます。
