## モード切替（自動モード） { #auto-mode }

Claude に「どこまで自動で実行させるか」を切り替える仕組みです。

- **都度確認（既定）** … 操作ごとに承認を求める。安全で初心者向き
- **自動モード** … 確認なしで連続実行。速いが、ファイル変更やコマンドを自動で行う

!!! warning "自動モードの注意"
    自動モードは確認なしで実行します。内容を把握できる作業・信頼できる範囲で使ってください。
