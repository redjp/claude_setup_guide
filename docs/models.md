# モデルと推論レベルの選び方

Claude は **モデル（頭脳の種類）** と **推論レベル（考える深さ）** を切り替えられます。「重要な仕事は賢いモデル＋深い思考」「日常は速いモデル」と使い分けると、**速さと精度のバランス**が取れます。

## 切り替えの場所

入力欄の**右下**に、現在のモデルと推論レベルが表示されています（例：**`Opus 4.8` `特大`**）。ここをクリックすると、選択メニューが開きます。

!!! note "🖼 画面イメージ（後日挿入予定）"
    入力欄右下のモデル／推論レベル選択メニュー（展開した状態）。

## モデルの選び方

モデルは「賢さ」と「速さ」のバランスが異なります。用途で選びましょう。

| モデル | 向いている用途 | イメージ |
| --- | --- | --- |
| **Opus** | 難しい判断・重要な文書・複雑な分析 | 一番賢い・じっくり |
| **Sonnet** | 日常の作業全般 | バランス型・実用的 |
| **Haiku** | 短い質問・素早い下書き | 軽量・高速 |

!!! tip "迷ったら"
    まずは標準のモデルのままでOK。**「これは大事」「難しい」**と感じたときに Opus に上げると失敗が減ります。

## 推論レベル（考える深さ）

推論レベルは、**Claude が答えを出す前に、内部でどれだけ手数をかけて検討するか** の設定です（例：**`特大`** は最も多く検討を重ねるモード）。手数を増やすほど、難しい問題での正確さが上がりますが、その分 **返答までの時間は長く** なります。「人が、即答するか・少し考えてから答えるか」の違いに近いものです。

- **高いレベル（特大など）** … 内部で多くの検討を重ねる → 正確だが時間がかかる → **重要な意思決定・込み入った分析**に
- **低いレベル（標準など）** … 検討は最小限で速い → **日常のやり取り・簡単な依頼**に

!!! note "速さ ↔ 精度のトレードオフ"
    「もっと慎重に考えてほしい」ときは上げ、「すぐ答えがほしい」ときは下げる、と覚えておけば十分です。

!!! note
    選べるモデルや推論レベルの**名称・上限は契約プランによって異なります**。メニューに表示されているものが、いま使える選択肢です。

## 使い分けの目安

- 役員会の資料・重要メール・契約の確認 → **Opus ＋ 深い推論**
- 議事録の要約・社内連絡の下書き → **Sonnet ＋ 標準**
- ちょっとした言い換え・確認 → **Haiku ＋ 標準**
