# モデルと推論レベルの選び方

「重要な仕事は賢いモデル＋深い思考」「日常は高速モデル」と使い分けると、速さと精度のバランスが取れます。

!!! info "このページは現在仕上げ中です"
    利用可能なモデル名・推論レベルの切替 UI を実機で確認し、スクリーンショットを添えて確定します。

## モデルの選び方

- **Opus** … 難しい / 重要な仕事
- **Sonnet** … 日常使い
- **Haiku** … 軽量・高速

## 推論レベル（拡張思考）

深く考えさせるモードの ON / OFF と、その使いどころ（速さ ↔ 精度のトレードオフ）。
