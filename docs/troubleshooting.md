# 困ったとき（FAQ）

よくあるつまずきと対処をまとめます。

## ログインできない

- メールアドレス／パスワードを確認してください。Google で登録した場合は **「Google でログイン」** を使います。
- パスワードを忘れた場合は、ログイン画面の「パスワードを忘れた場合」から再設定します。
- それでも入れない場合は、一度アプリを完全に終了し、再起動してから試してください。

## データ連携（Connectors）がつながらない

- 連携時の **Microsoft サインインで「許可」** を押したか確認してください。
- 会社のアカウントは、**IT 管理者の承認が必要**な場合があります。情報システム担当にご相談ください。
- 一度「接続を解除」してから、つなぎ直すと改善することがあります。
- 詳しくは [データ連携（Connectors）](connectors.md) を参照。

## 動作が遅い・重い

- **推論レベル**を下げる、または軽いモデル（Sonnet / Haiku）に切り替えると速くなります → [モデルと推論レベルの選び方](models.md)
- 長い会話は動作が重くなりがちです。区切りのよいところで **「新規チャット」** を開くと軽くなります。
- 大きなファイルを添付すると時間がかかります。

## モデルや機能が見当たらない

- 使えるモデル・推論レベル・連携は **契約プランによって異なります**。
- 上限に達して一時的に使えないこともあります。時間をおくか、プラン（Pro / Max）をご確認ください。
- アプリのバージョンが古いと表示が異なる場合があります。最新版に更新してください。

## 機密情報を入力してしまった／不安なとき

- パスワードや口座番号など、極めて機密性の高い情報は**入力しない**のが基本です。
- 会話を学習に使わない設定は [プライバシー・データ利用](settings.md) で確認できます。
- 不要な会話は履歴から削除できます。

## それでも解決しないとき

- アプリを再起動する／PC を再起動する
- アプリを最新版に更新する
- 公式ヘルプ（[https://support.claude.com/](https://support.claude.com/)）を確認する
- 社内の情報システム担当に相談する
