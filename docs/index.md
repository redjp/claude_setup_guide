# はじめに

このガイドは、**Claude Desktop（クロード・デスクトップ）を新しく使い始める方**のための、最短で迷わないセットアップ＆活用ガイドです。専門知識は不要です。上から順に読まなくても、必要なところだけ拾い読みできます。

## Claude とは

Claude（クロード）は、文章の作成・要約・翻訳・相談・調査・資料づくりなどを、**日本語で対話しながら任せられる AI アシスタント**です。Claude Desktop は、それを PC のアプリとして快適に使えるようにしたものです。

たとえばこんなことができます。

- メールや議事録、提案書のたたき台を作る
- 長い資料や PDF を要約して、要点だけ受け取る
- 手元のファイルや社内データ（Microsoft 365 など）とつないで質問する
- 表計算・スライド・コードなど、実作業そのものを任せる

## 主な用語の関係（先に知っておくと迷いません）

Claude には似た言葉が出てきます。関係を一度つかんでおくと、後の章で迷いません。

- **Connectors（連携）** … Claude と外部サービス（Microsoft 365 など）を**つなぐ配管**。自分のデータを参照させる仕組み。→ [データ連携](connectors.md)
- **スキル（Skills）** … 特定の作業手順をパッケージ化した**個別の「技」**。→ [スキル](skills.md)
- **プラグイン** … 役割・分野ごとに、関連するスキルや設定をまとめた**道具箱（パック）**。
- **Cowork** … それらの道具を使って、実務や成果物を仕上げる**作業モード**。→ [Cowork](cowork.md)
- **MCP** … 外部ツールと Claude をつなぐ技術的な仕組み（連携の“裏側”の規格）。**上級者向け**で、通常は意識不要。

!!! note "この環境について"
    本ガイドは **Windows 11** の Claude Desktop を前提に作成しています。ログインは Claude のサブスクリプション（契約プラン）で行い、**開発者向けの API キーは不要**です。
