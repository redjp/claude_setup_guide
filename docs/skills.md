# スキル（Skills）

スキルは、特定の作業手順や専門知識をパッケージ化して Claude に追加できる仕組みです。便利な反面、**導入時の安全確認が大切**です。

!!! info "このページは現在仕上げ中です"
    `grill-me` スキルを例に、安全な導入手順を実機で確認して確定します。

## 安全に導入する手順（例：grill-me）

!!! warning "導入前に必ず中身を確認"
    スキルは Claude に指示や手順を与えるものです。配布元が信頼できるか、`SKILL.md` の中身に不審な操作がないかを、**有効化する前に**確認します。
